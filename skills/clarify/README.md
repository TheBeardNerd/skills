# clarify

Use when the user asks to clarify, sharpen, or de-vague a prompt, or when a request is genuinely ambiguous or under-specified. Interviews the user with structured multiple-choice questions to pin down intent, scope, and success criteria, rewrites the prompt, confirms it, then proceeds. Also invoked automatically by a UserPromptSubmit hook on ambiguous prompts.

## Install

```bash
npx skills add https://github.com/TheBeardNerd/skills --skill clarify
```

## Usage

| Intent | Example prompt |
|--------|----------------|
| <intent> | `"<example>"` |
